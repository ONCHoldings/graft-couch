graft-couch
===========

couchdb data adaptor for couchdb

### Run tests

You must configure CouchDB to work with HTTPS in order to run tests.
See [this wiki page](http://wiki.apache.org/couchdb/How_to_enable_SSL)

    cd graft-couch/
    make simple
